# Execution Checklist

Track progress with this checklist.

## A) Setup

- [ ] Python installed and verified.
- [ ] Node.js and npm installed and verified.
- [ ] Backend dependencies installed in `.venv`.
- [ ] Frontend dependencies installed.

## B) Baseline Runtime

- [ ] Backend starts successfully.
- [ ] Frontend starts successfully.
- [ ] Smoke test script passes.

## C) Data and Modeling

- [ ] Dataset inventory validated.
- [ ] Tabular baseline trained.
- [ ] Baseline metrics exported.

## D) API Integration

- [ ] `/endo/train` implemented.
- [ ] `/endo/predict` implemented.
- [ ] API contract documented and validated.

## E) Frontend Integration

- [ ] Train page wired to backend.
- [ ] Predictions page wired to backend.
- [ ] Error/loading states handled.

## F) Final Packaging

- [ ] Report generated.
- [ ] Metrics file generated.
- [ ] Model artifact saved.
- [ ] Decision log updated.
